# Alodoc

Learn. Understand. Live well.  
Belajar. Pahami. Hidup sehat.

Alodoc is a bilingual health education dApp for common disease learning cards, quizzes, and onchain proof-of-learning badges on Arc Testnet.

It is not a diagnosis app, doctor consultation app, symptom checker, or medical record app.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- wagmi + viem

## Arc Testnet

- Chain ID: `5042002`
- RPC URL: `https://rpc.testnet.arc.network`
- Native currency: `USDC`
- Explorer: `https://testnet.arcscan.app`
- Contract: `0x6eDc6a30D2735E71afDB622026a46343e6dD81fa`
